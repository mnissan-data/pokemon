# Pokémon Binder Tracker

An interactive, single-file web app for tracking a vintage Pokémon card collection across the **Base Set, Jungle, and Fossil** sets (all 228 cards).

## Features
- **9-pocket binder view** with real card images.
- **Sort modes:** Type/Color, National Dex, Set + #, Rarity, Evolution Family (horizontal), and Evo Vertical (evolution lines as columns).
- **Live metrics:** overall completion, per-set %, rarity breakdowns (holo/rare/uncommon/common), and original-151 progress.
- **Click to toggle** owned/needed — progress saves in your browser (localStorage).
- **TCG energy-type icons** and the Pokémon color palette.
- **Mobile-friendly** responsive layout.

## Usage
Open `index.html` (or `pokemon_binder.html`) in any browser. To display the official logo in the header, place a `pokemon_logo.png` file in the same folder.
